## 📝 Descripción
Este proyecto consiste en una aplicación de gestión bibliotecaria desarrollada para el módulo de **Programación**. El software permite administrar de forma integral el catálogo de libros, los registros de usuarios y la lógica de préstamos y devoluciones.

Es una evolución de las versiones anteriores, optimizando la estructura de clases y mejorando la persistencia de los datos.

---

## ✨ Funcionalidades
*   **Gestión de Inventario:** Altas, bajas y modificaciones de libros.
*   **Control de Socios:** Registro y consulta de los usuarios de la biblioteca.
*   **Sistema de Préstamos:** Gestión de disponibilidad de ejemplares y fechas.
*   **Persistencia de Datos:** Almacenamiento en archivos para mantener la información tras el cierre del programa.

## 🛠️ Conceptos Aplicados
En esta tarea se han puesto en práctica los siguientes pilares:
- **POO:** Herencia, polimorfismo y encapsulamiento.
- **Estructuras de Datos:** Uso de `Collections` (Listas, Mapas o Conjuntos).
- **Manejo de Excepciones:** Robustez ante errores de entrada o de archivos.
- **E/S de Ficheros:** Lectura y escritura para persistencia.
